# 🏟️ Sports Management System (MERN)

A full-stack Sports Management System developed using the **MERN stack (MongoDB, Express.js, React, Node.js)**.  
The application helps manage sports grounds, bookings, training sessions, and user operations with role-based access.

---

## 🚀 Key Features

- Secure login & registration
- Role-based dashboard
- Ground booking & scheduling
- Training session management
- User management
- Booking approvals
- Reports and monitoring
- Responsive design

---

## 🧰 Technology Stack

### Frontend
- React.js
- Axios
- Bootstrap / CSS / Tailwind (edit based on yours)

### Backend
- Node.js
- Express.js

### Database
- MongoDB

### Authentication
- JSON Web Token (JWT)

---

## 👥 System Roles & Responsibilities

### 🛡️ Admin
- Manage all users
- Assign roles
- Approve or reject bookings
- View system reports
- Maintain platform settings

### 🏃 Player
- Register and login
- View ground availability
- Make bookings
- View booking history
- Get notifications

### 🧑‍🏫 Coach
- Create and manage training sessions
- View assigned players
- Request ground reservations
- Track schedules

### ⚙️ Operation Manager
- Supervise daily activities
- Manage schedules
- Allocate resources
- Generate reports

### 🌱 Ground Manager
- Update ground availability
- Handle maintenance
- Approve or cancel reservations
- Monitor usage
